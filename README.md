# Port Intent Detector PRO

A professional-grade network **Port Intent Detector** with:

- Real-time outbound connection monitoring
- CLI, Desktop GUI, and Web Dashboard
- Auto PDF/CSV reporting
- Alerts for suspicious ports
- Dark-mode web UI with live charts
- Notification system and auto-updater

---

## Features

- CLI monitor with live intent
- Desktop GUI (PyQt6) for real-time monitoring
- Web dashboard (Flask + SocketIO) with charts and dark mode
- Custom port-intent mapping
- Auto-report generation (PDF/CSV)
- Alerts for unknown or suspicious connections
- Works cross-platform (Linux, Windows, macOS)
- Installable via pip from GitHub

---
## Requirements
Python 3.8+
psutil, Flask, Flask-SocketIO, PyQt6, plotly, reportlab

Usage
## CLI Monitor
port-intent-pro

## Web Dashboard
